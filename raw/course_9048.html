Remark
Teaching goal 1.Regardless of programming background, anyone can easily get started using app development tools that utilize a visual, block-based interface to create apps. 2.Become familiar with visual app development tools and their related features, enabling the development of Android apps and cultivating mobile application programming skills. 3.Learn programming concepts and develop the ability to use code to solve problems, laying a solid foundation for future programming courses. 4.Learn how to observe and define problems through design thinking, and stimulate cross-disciplinary creativity and problem-solving abilities. 5.Understand the core concepts of design thinking, explore real-life issues, identify their root causes, propose innovative design solutions, and experience the key steps and essence of design thinking through hands-on implementation and validation.

W14. App Project Testing and Refinement / Testing in Design Thinking
W15. App Project Testing and Refinement / Testing in Design Thinking
