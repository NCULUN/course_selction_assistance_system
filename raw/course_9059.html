Remark
Teaching goal 1. Learn programming concepts 2. Learn program conditional logic concepts 3. Learn programming loop structures 4. Learn to define program function concepts 5. Cultivate the ability to develop simple programs 6. Cultivate computational thinking skills
Teaching content 1. Programming concepts: introduction to input and output functions
2. Variable data types: numerical data type (int, float), string data type (String), container data type (List, Dictionary)
3. Programming logic: if, if...else, if...elif...else, nested if
4. Programming logic: for loop, while loop
5. Function concepts and definitions (def)
6. Program project topic: Chess
