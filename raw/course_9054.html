Remark
Teaching goal 1. Be able to write programs in Python for data cleaning, processing, and analysis. 2. Be proficient in using Pandas for data manipulation (importing, filtering, aggregation, transformation, etc.). 3. Be able to present data analysis results in charts and other visualizations to enhance data interpretation skills. 4. Be able to design and implement basic machine learning models (such as clustering and regression) and perform model evaluation. 5. Through midterm exams, final exams, and final project, be able to integrate all knowledge learned and complete small data analysis and forecasting projects.
